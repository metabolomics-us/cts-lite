## CTS-Lite Dataset
This directory stores the dataset and the tools used to create it

### Dataset Creation
Simply run the `create_csv_dataset.sh` script found under `cmd/`, then follow the instructions in the root's README to build the base image

### Dockerfile
The Dockerfile in this directory is used to create the app's base image, containing only the dataset

### cmd
The cmd directory holds helper programs and scripts used to construct the dataset and database

# CTSLite

A lightweight chemical translation service using the PubChemLite dataset, written in Go

### Notes

- CTSLite was written in Go 1.23.1
- The PubChemLite dataset can be download [here](https://zenodo.org/records/17076905)
- To use the dataset trimmer you will need csvkit, install it with `sudo apt install csvkit`
- Load tests are performed using the Locust python framework found [here](https://locust.io/)
